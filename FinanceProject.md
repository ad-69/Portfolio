# AI-Based Stock Trading Simulation

A beginner Python project inspired by the research paper:

"TradingAgents: Multi-Agents LLM Financial Trading Framework"

The project simulates how different AI trading agents collaborate to make trading decisions.

---

## Research Inspiration: 

This project was inspired by the paper:
https://arxiv.org/pdf/2412.20138

The original paper proposes a collaborative multiagent framework involving:

- Technical Analysis Agent
- Sentiment Analysis Agent
- Fundamental Analysis Agent
- Trader Decision Agent
- Risk Management System

I attempted to implement a simplified version of this architecture using Python classes and Object-Oriented Programming (OOP) concepts before integrating APIs or real-time financial data.

The paper also highlights two major challenges in existing trading systems:

1. A single model performing multiple roles inefficiently, or multi-agent systems independently gathering overlapping data.
2. Lack of structured communication due to heavy reliance on natural language interactions.

This project explores these ideas through a basic modular agent-based simulation.
